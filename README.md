# Buckshot Roulette Clone

Учебный проект — собственная реализация игры, вдохновлённой Buckshot Roulette, на языке Go с использованием Ebitengine.

## Цели проекта

- изучить основы языка Go;
- освоить работу с Ebitengine;
- освоить работу с 2D-графикой, звуками и анимациями;

## Планируемые возможности

- главное меню;
- пошаговая система ходов;
- боевые и холостые патроны;
- здоровье игрока и дилера;
- случайное поведение дилера;
- предметы;
- анимации и звуковые эффекты;
- экран победы и поражения;
- перезапуск игры.

## Технологии

- Go
- Ebitengine
- OpenGL**
- C/C++**
